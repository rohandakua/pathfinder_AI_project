
# How to use :
Key | Usage
--------------|--------------
a | A-Star algorithm
b | Breadth-First-Search
d | Depth-First-Search
v | Best-First-Search
h | HillClimb
n | Djikstra
g | Generate maze (DFS + Backtracker)
Backspace | Clear Screen
s | set starting point
e | set end point
leftClick | Make wall
rightClick | Remove Wall
